# Resumo Científico

O presente trabalho descreve o FallSense, um sistema integrado de detecção de quedas voltado para o monitoramento de idosos. O objetivo principal é proporcionar uma resposta rápida a eventos críticos de queda por meio de uma arquitetura composta por uma pulseira IoT, uma API backend e um aplicativo móvel. A metodologia técnica baseia-se na utilização do microcontrolador XIAO ESP32-C3 integrado ao sensor de movimento MPU6050 (acelerômetro e giroscópio). O algoritmo de detecção, executado diretamente no firmware, opera em duas fases: cálculo contínuo do Signal Vector Magnitude (SVM) para detecção de impacto, utilizando um limite de 2.5G, e verificação de postura via ângulos de pitch e roll. Para mitigar falsos positivos, o sistema inclui uma janela de cancelamento manual de 10 segundos. A transmissão de dados para o backend em FastAPI e PostgreSQL ocorre sob o protocolo MQTT sobre TLS, enquanto o aplicativo desenvolvido em Flutter consome a API via conexões seguras HTTPS. O sistema implementa rigorosos mecanismos de segurança, incluindo o hash de senhas utilizando o algoritmo Argon2id, criptografia simétrica Fernet (AES-128-CBC) para a proteção de dados em repouso, e controle de sessão através de JSON Web Tokens (JWT - HS256) com mecanismo de revogação (blacklist). A autenticação de dois fatores (2FA) via TOTP e a proteção ativa contra ataques de força bruta reforçam o controle de acesso à plataforma. Em conformidade com a Lei Geral de Proteção de Dados (LGPD), o sistema garante o princípio da minimização de dados, exige consentimento explícito durante o cadastro, assegura a criptografia de dados sensíveis, mantém a rastreabilidade completa por meio de logs de auditoria e aplica políticas bem definidas para a retenção e exclusão de informações, protegendo assim a privacidade e a integridade dos usuários monitorados.
