# Backlogs e QA

Pasta para centralizar materiais de planejamento, backlog e validação manual do projeto.

## Arquivos

- `BACKLOG_PERFIL_APP.md`: backlog incremental das funcionalidades de perfil.
- `QA_PERFIL_APP.md`: checklist de validação manual do fluxo de perfil.

## Convenção

- Backlogs: `BACKLOG_<AREA>.md`
- QA/checklists: `QA_<AREA>.md`

Manter esses arquivos aqui evita espalhar documentos operacionais na raiz do repositório.
